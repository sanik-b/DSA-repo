##  About

This repository contains my solutions to Data Structures and Algorithms problems from various coding platforms. All solutions are **automatically synced** using [LeetSync](https://github.com/LeetSync/LeetSync) - a browser extension that pushes accepted code directly to GitHub.

**Why this repository?**
-  **Auto-Synced** - No manual uploads, just solve and commit
-  **Track Progress** - Dashboard shows solved count by difficulty
-  **Rich Metadata** - Each solution includes problem description, tags, runtime, and memory stats
-  **Multi-Platform** - LeetCode, GeeksforGeeks, Codeforces, and more

## 🛠️ How It Works

### LeetSync Automation
1. Solve a problem on LeetCode/GFG/Codeforces
2. Get **Accepted** ✅
3. LeetSync automatically:
   - Creates problem folder
   - Saves your solution
   - Generates README with stats
   - Pushes to GitHub

## 🚀 Quick Start

### Prerequisites
- [LeetSync Extension](https://chrome.google.com/webstore/detail/leetsync)
- GitHub Account
- LeetCode/GFG Account

### Setup
1. Clone this repository:
```bash
git clone https://github.com/yourusername/dsa-repo.git
cd dsa-repo
```

2. Configure LeetSync:
   - Click extension icon
   - Enter GitHub token
   - Select repository and branch
   - Choose languages

3. Start solving! 🎯
